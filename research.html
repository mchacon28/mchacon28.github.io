Research

Research Areas

My research focuses on developing topological methods for complex data analysis and machine learning. Specifically, I work on:

  • Neural networks with topological features
  • Biomedical data analysis using persistent homology
  • Reinforcement learning algorithm optimization
  • Topological data visualization

Theoretical Framework

My work is based on persistent homology theory and algebraic topology, applying these concepts to practical problems in machine learning. For example:

\[ H_k(X) = \ker(\partial_k) / \text{im}(\partial_{k+1}) \]

Current Projects

  • Biomedical data analysis using persistent homology techniques
  • Optimization of reinforcement learning algorithms with topological features
